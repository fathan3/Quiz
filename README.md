# Mini Quiz - OOP Implementation
Interactive Flashcard Quiz built with Python Flask and OOP Principles.

## Fitur Utama
- **OOP Architecture:** Penggunaan class untuk manajemen data dan logika.
- **Admin Dashboard:** CRUD soal kuis secara aman.
- **User Interface:** Sistem kuis interaktif dengan skor real-time.

## Tech Stack
- Python (Flask Framework)
- MySQL
- HTML/CSS (Jinja2 Templates)
